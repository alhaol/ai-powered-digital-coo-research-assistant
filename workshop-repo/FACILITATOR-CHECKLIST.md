# ✅ Facilitator Checklist

## One Week Before
- [ ] Test all CLI commands on both macOS and Windows (WSL/PowerShell)
- [ ] Generate fresh sample outputs for `sample-outputs/`
- [ ] Verify `datasets/sample_survey_responses.csv` loads correctly in the AI CLI
- [ ] Prepare backup screenshots/recordings of each exercise (Wi-Fi insurance)
- [ ] Print 10% extra booklets
- [ ] Test QR codes
- [ ] Verify all reference commands in `reference-commands/` work with Claude Code
- [ ] Verify the reference skill in `reference-skills/lean-startup/` auto-triggers
- [ ] Test the full command-building flow on a clean repo clone (simulate participants)
- [ ] Prepare the live-demo command: copy `reference-commands/analyze-survey.md`
      into `.claude/commands/` for the opening demo, then remove it before participants clone

## Day Of
- [ ] Arrive 30 min early for A/V setup
- [ ] Terminal font size 20+ for readability
- [ ] Pre-load the repo in your terminal (`cd` into it)
- [ ] Have the CSV, `competitors.md`, and `CLAUDE.md` ready
- [ ] Test Wi-Fi and API connectivity
- [ ] Display the repo QR during setup time
- [ ] Have a "Plan B" deck with pre-recorded terminal GIFs (API-outage insurance)
- [ ] Confirm `.claude/commands/` and `.claude/skills/` exist but are empty (only `.gitkeep`)
- [ ] Have `reference-commands/` ready as a fallback for stuck participants

## Post-Workshop
- [ ] Push live-generated artifacts to a `live-demo-outputs/` branch
- [ ] Send follow-up email within 24h (booklet PDF, repo link, community invite)
- [ ] Collect feedback via a short survey (≤ 5 questions)
- [ ] Post takeaways + photos with the event hashtag
- [ ] Include Claude Code commands/skills docs links in the follow-up
- [ ] Share a "Command of the Week" for the 4 weeks after the workshop

---

## The Single Most Important Moment
The **opening live demo** (Slide 5). Rehearse it until it's flawless. Use the exact
CSV from the repo. The audience must *feel* the speed — messy CSV to formatted
report in ~30 seconds — then see the same result from a one-line command. That
"wow" earns you the next 85 minutes.
