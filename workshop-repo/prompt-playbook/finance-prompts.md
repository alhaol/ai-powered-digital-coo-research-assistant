# Finance Prompts

Five reusable prompts for financial modeling and reporting.

## 1. 12-Month Financial Projection
```
You are a startup CFO. Build a simple 12-month P&L projection for [business]
assuming [starting revenue], [growth rate], [COGS %], and [monthly fixed costs].
Output a month-by-month markdown table (revenue, COGS, gross profit, opex, net)
and note the month I reach breakeven. State every assumption.
```

## 2. Unit Economics
```
You are a unit-economics analyst. For [product] at [price] with [COGS], [CAC],
and [expected repeat rate], calculate contribution margin, LTV, LTV:CAC ratio,
and payback period. Show the math. Flag whether the model is healthy and the one
lever that most improves LTV:CAC.
```

## 3. Burn Rate & Runway
```
You are a finance advisor. Given [cash on hand], [monthly burn], and [expected
monthly revenue growth], calculate current runway in months and the runway if I
cut burn by 20%. Show a 6-month cash balance table and the date I must raise or
be profitable by.
```

## 4. Pricing Model Comparison
```
You are a pricing analyst. Compare 3 models for [product]: one-time purchase,
subscription, and hybrid. For each: revenue predictability, CAC payback, customer
friction, and margin. Recommend one for a bootstrapped stage and explain the
trade-off I'm accepting.
```

## 5. Monthly Investor Update
```
You are a founder writing to investors. Draft a concise monthly update for
[startup] covering: TL;DR, key metrics vs. last month, wins, lowlights, asks, and
runway. Keep it under 400 words, honest, and skimmable. Use the metrics I provide:
[metrics].
```
