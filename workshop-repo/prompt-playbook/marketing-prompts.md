# Marketing Prompts

Five reusable prompts for marketing and growth.

## 1. 30-Day Social Content Calendar
```
You are a social media strategist. Create a 30-day content calendar for [brand]
on [platform], targeting [audience]. Mix educational, behind-the-scenes,
social-proof, and CTA posts (rough 40/20/20/20 split). Output a markdown table:
Day, Theme, Hook, Format, CTA. Keep the brand voice [tone].
```

## 2. Email Welcome Sequence
```
You are an email marketer. Write a 5-email welcome sequence for new [waitlist/
customers] of [brand]. For each email: goal, subject line (2 variants), preview
text, and body copy. Sequence should move from welcome → story → education →
social proof → offer. Tone: [tone].
```

## 3. Ad Copy Variants
```
You are a direct-response copywriter. Write 5 ad variants for [product] aimed at
[audience] on [platform]. Each variant: a distinct angle (pain, aspiration,
proof, curiosity, urgency), a headline, primary text, and CTA. Note which angle
you'd test first and why.
```

## 4. SEO Blog Post Brief
```
You are an SEO content strategist. For the keyword "[keyword]", produce a blog
post brief: search intent, suggested title (3 options), H2/H3 outline, target
word count, internal-link ideas, and the one question the post must answer better
than page-one competitors.
```

## 5. Launch Announcement
```
You are a PR and comms lead. Draft a launch announcement for [product] in three
formats: (1) a 280-char social post, (2) a 150-word LinkedIn post, (3) a 5-line
press blurb. Lead with the customer benefit, not the company. Tone: [tone].
```
