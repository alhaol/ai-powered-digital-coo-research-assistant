# Operations Prompts

Five reusable prompts for running the operational grind.

## 1. Standard Operating Procedure (SOP)
```
You are an operations manager. Write a detailed SOP for [process]. Include: title
+ version, purpose/scope, numbered steps, if/then decision points, email
templates, escalation criteria, and KPIs. Format so a new hire could follow it on
day one.
```

## 2. New Hire / Contractor Onboarding Checklist
```
You are a people-ops lead. Create an onboarding checklist for a new [role] at
[company]. Group by: before day 1, day 1, week 1, month 1. Include access/tools,
context docs to read, first task, and a 30-day success definition.
```

## 3. Vendor / Supplier Evaluation
```
You are a procurement analyst. Build a scoring rubric to evaluate suppliers for
[input]. Define 6 criteria (e.g. quality, price, sustainability, reliability, MOQ,
lead time) with weights. Output a scorecard template and the 8 questions I should
ask every supplier.
```

## 4. Meeting Agenda + Notes Template
```
You are a chief of staff. Create a reusable agenda template for a [meeting type].
Include: objective, pre-reads, timeboxed sections, decision log, and action items
(owner + due date). Keep it to one page.
```

## 5. Weekly Ops Dashboard
```
You are an operations analyst. Design a weekly ops dashboard template for
[business] that I fill in every Friday. Include the 6-8 metrics that matter most
at my stage (e.g. CAC, LTV, refund rate, inventory, cash, NPS), each with a target
and a "trend vs last week" column. Markdown tables only.
```
