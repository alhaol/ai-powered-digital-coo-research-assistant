# Strategy Prompts

Five reusable prompts for strategic decisions. Swap the **[bracketed]** parts.

## 1. Market Sizing (TAM / SAM / SOM)
```
You are a market analyst. Estimate the TAM, SAM, and SOM for [product] targeting
[segment] in [geography]. Show your assumptions and the arithmetic for each layer.
Give a low/base/high range and flag the single assumption that most affects the
result. Format as a markdown table plus a 100-word interpretation.
```

## 2. Positioning Statement
```
You are a brand strategist. Write a positioning statement for [company] using the
format: "For [target] who [need], [product] is a [category] that [key benefit],
unlike [main competitor], because [reason to believe]." Then give 3 alternative
angles and recommend one with justification.
```

## 3. Pricing Strategy
```
You are a pricing strategist for a bootstrapped startup. Propose 3 pricing tiers
for [product]. For each: price point, target buyer, included features, and the
psychological anchor it sets. Assume [constraint, e.g. sub-$50, 12% premium].
Recommend a launch price and explain the margin implication.
```

## 4. Partnership Evaluation
```
You are a corporate development advisor. I'm considering a partnership with
[partner] to [goal]. List the strategic rationale, what each side gives and gets,
the top 3 risks, and 5 questions I must answer before signing. Conclude with a
go / no-go recommendation and confidence level.
```

## 5. Fundraising Readiness
```
You are a pre-seed investor. Assess whether [startup] is ready to raise, given:
[traction, team, market]. Score readiness (0-10) on team, market, traction,
product, and moat. List the 3 weakest areas and exactly what would move each up
one point. Be blunt.
```
